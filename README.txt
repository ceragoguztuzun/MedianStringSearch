To compile and run the file: 

Be in the directory where Makefile, hwbs.c and input.txt files are in.
Then in the command line type "make"
Then execute using "./hwbs input0.txt"

This will print the results in the command line as well as create a file named "output.txt" in which the best pattern will be written.
